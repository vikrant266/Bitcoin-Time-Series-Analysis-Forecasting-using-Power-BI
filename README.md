# 📊 Bitcoin Time Series Analysis & Forecasting using Power BI

## 🚀 Project Overview
This project focuses on analyzing Bitcoin historical data using **SQL and Power BI**.  
The goal is to understand market trends, volatility, and perform forecasting using interactive dashboards.

The project follows an end-to-end data analysis pipeline:
> SQL → Data Cleaning → DAX → Visualization → Forecasting → Insights

---

## 📂 Dataset
The dataset contains Bitcoin historical data with the following fields:

- Date  
- Open Price  
- High Price  
- Low Price  
- Close Price  
- Volume  

---

## 🛠️ Tools & Technologies Used

- **SQL (PostgreSQL / SQL Server)** → Data Cleaning & Analysis  
- **Power BI** → Data Visualization & Dashboard  
- **DAX** → Calculated Measures  
- **Power Query** → Data Transformation  

---

## ⚙️ Project Workflow

### 🔹 1. Data Cleaning using SQL
- Removed null values  
- Sorted and validated data  
- Performed basic analysis (avg, max, min, volume)  
- Created return calculations for volatility  

---

### 🔹 2. Data Transformation (Power Query)
- Changed data types  
- Handled missing values  
- Structured dataset for analysis  

---

### 🔹 3. Feature Engineering (DAX)
Created important measures like:
- Average Price  
- Maximum & Minimum Price  
- Total Volume  
- Price Change  
- Moving Averages (MA7, MA30)  

---

### 🔹 4. Dashboard Development (Power BI)

#### 📌 Page 1: Executive Overview
- KPI Cards (Avg, Max, Min, Volume)  
- Price Trend (Line Chart)  
- Volume Analysis  

#### 📌 Page 2: Price Analysis
- Closing Price Trend  
- Open vs Close Comparison  
- High vs Low Analysis  

#### 📌 Page 3: Forecast & Trend
- Price Forecast (with confidence interval)  
- Moving Average (MA7 & MA30)  
- Trend Direction  

---

## 📊 Key Insights

- Bitcoin shows an overall **upward trend**  
- Market is **highly volatile**  
- Trading volume strongly impacts price  
- Forecast indicates possible future growth  
- Moving averages help identify trend direction
